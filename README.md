# Сервис коротких ссылок + QR (Yii2 Basic)

Тестовое задание реализовано на стеке:
- Yii2 Basic
- MySQL
- jQuery
- Bootstrap

## Что реализовано

- Форма на главной странице (`/`) с полем URL и кнопкой `OK`.
- Валидация URL (`http`/`https`).
- Проверка доступности ресурса (cURL HEAD-запрос).
- При недоступности выводится ошибка: `Данный URL не доступен.`
- При успехе:
  - сохранение ссылки в БД,
  - генерация **уникального** `short_code`,
  - генерация короткой ссылки,
  - генерация QR-кода.
- Ответ фронтенду через Ajax без перезагрузки страницы.
- Переход по короткой ссылке `/r/{code}` делает редирект на оригинальный URL.
- При редиректе пишется лог перехода (IP, user-agent, referer) и увеличивается счётчик переходов.
- Публичная страница статистики `/stats`:
  - поиск по `short_code`,
  - поиск по IP,
  - фильтр по минимальному количеству переходов с IP,
  - таблица с short code, IP, количеством переходов с IP, общим количеством переходов и временем последнего перехода.

## Важно про `short_code`

`short_code` сделан уникальным на двух уровнях:
1. На уровне БД: уникальный индекс `ux_links_short_code`.
2. На уровне модели Yii2: правило `unique`.

## Структура БД

Используются таблицы:
- `links` — исходный URL, short_code, счетчик кликов и статусы.
- `link_click_logs` — логи переходов с внешним IP и метаданными.

Создание схемы выполняется миграцией:
- `app/migrations/m260410_000001_create_links_and_link_click_logs_tables.php`


## Нужен ли Docker, если уже есть Nginx на машине?

Коротко: **нет, Docker не обязателен**.

Если у вас уже установлен Nginx + PHP 8.4 + MySQL/MariaDB, проект можно запускать без Docker:

1. Установить зависимости в `app/`:
   ```bash
   composer install
   ```
2. Настроить БД в `app/config/db.php` (или через env-переменные `DB_*`).
3. Выполнить миграции:
   ```bash
   php yii migrate --interactive=0
   ```
4. В Nginx указать root на `app/web` и направлять запросы на `index.php` (через `try_files`).

Docker в этом репозитории дан как **удобный воспроизводимый вариант** развёртывания для проверки.

## Запуск через Docker (Nginx + PHP-FPM 8.4 + MySQL 8.4)

Подготовлены контейнеры с именами:
- `nginx_test_ayan`
- `php_test_ayan`
- `mysql_test_ayan`

### 1) Перейти в каталог приложения

```bash
cd app
```

### 2) Собрать и запустить контейнеры

```bash
docker compose up -d --build
```

### 3) Установить зависимости Composer

```bash
docker compose exec php_test_ayan composer install
```

### 4) Применить миграции

```bash
docker compose exec php_test_ayan php yii migrate --interactive=0
```

### 5) Открыть проект

- Основной интерфейс: http://localhost:8080

## Полезные команды

Остановить контейнеры:

```bash
docker compose down
```

Остановить и удалить volume MySQL:

```bash
docker compose down -v
```
